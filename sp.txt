Scapy (Python Networking) 
 A powerful Python tool for packet creation, sending, and sniffing.  
 Used for network analysis, scanning, and security testing.  
 Helps build custom packets like ARP, ICMP, TCP.

How to sniff with Scapy (capture 10 packets and show details of all packets)?
Code
from scapy.all import *
packets = sniff(count=10)
packets.summary()

    pkt.show()

 4-a) ARP Network Scanner
Write a Python code using Scapy for creating a Network Scanner (Hint: Create an ARP packet using ARP() method).
Code
import scapy.all as scapy
request = scapy.ARP(pdst="192.168.1.1/28")
broadcast = scapy.Ether(dst="ff:ff:ff:ff:ff:ff")
request_broadcast = broadcast / request
clients = scapy.srp(request_broadcast, timeout=2)[0]
for element in clients:
    print(element[1].psrc + " - " + element[1].hwsrc)
Wireshark Step
 Filter:arp

Sends ARP requests to all devices in the network. Active devices reply with IP and MAC addresses. This helps identify connected systems.

 8-a) Port Scan using Scapy
Demonstrate how to perform a Port Scan using Scapy tool on target www.example.com
 Code
from scapy.all import *
target = "www.example.com"
for port in range(20, 25):
    packet = IP(dst=target)/TCP(dport=port, flags="S")
    response = sr1(packet, timeout=1, verbose=0)
    if response:
        if response.haslayer(TCP) and response.getlayer(TCP).flags == 18:
            print("Port", port, "is open")
Wireshark Step
Filter:tcp

Sends TCP SYN packets to check if ports are open. If the response is SYN-ACK, the port is open. This method is used for basic port scanning.

10-a) ICMP Monitor (Callback)
Create a simple network monitor that watches for ICMP packets using Python function callback in Scapy.
Code
from scapy.all import *
def callback(packet):
    if packet.haslayer(ICMP):
        print("ICMP Packet Detected")
        packet.show()
sniff(filter="icmp", prn=callback, count=10)
Wireshark Step
 Filter:icmp

Monitors network traffic and prints only ICMP packets. The callback function processes each captured packet. Useful for detecting ping activity.
