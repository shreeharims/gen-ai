BadStore (Web Security / SQL Injection / Cookies) 
• A vulnerable web application used for learning security concepts.

Scapy (Python Networking) 
• A powerful Python tool for packet creation, sending, and sniffing.  
• Used for network analysis, scanning, and security testing.  
• Helps build custom packets like ARP, ICMP, TCP.  
NetCat (Networking Tool) 
• A simple tool for sending and receiving data over network connections.  
• Used for port scanning, chatting between systems, and debugging.  
• Known as the “Swiss army knife of networking”.  
Nmap (Scanning Tool) 
• A tool used to discover hosts and open ports in a network.  
• Helps in security auditing and vulnerability detection.  
• Can perform port scan, OS detection, and service version detection.

17. Full Question 
Write a Shell script for port scan using NetCat. 
Important Terms 
• Shell Script → A file with multiple Linux commands 
• Port Scan → Checking which ports are open 
• NetCat (nc) → Tool to connect/read/write network data 
• -z → Scan mode (no data transfer) 
• -v → Verbose (shows result) 
What you will get as Output 
• List of ports: 
o Open 
o Closed 
Demo 
for port in {20..25} 
do 
nc -zv 127.0.0.1 $port 
done 
Output Example 
Connection to 127.0.0.1 22 port [tcp/ssh] succeeded! 
nc: connect to 127.0.0.1 port 23 failed 
 
 What you say 
“Here I wrote a shell script using NetCat to scan ports from 20 to 25. The -z option 
checks port status without sending data, and -v shows detailed output. It helps identify 
which ports are open on a system.”
