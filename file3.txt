BadStore (Web Security / SQL Injection / Cookies) 
• A vulnerable web application used for learning security concepts.  
• Helps practice SQL Injection, cookie manipulation, authentication bypass.  
• Used to understand how attackers exploit web application weaknesses.  
Scapy (Python Networking) 
• A powerful Python tool for packet creation, sending, and sniffing.  
• Used for network analysis, scanning, and security testing.  
• Helps build custom packets like ARP, ICMP, TCP.  
NetCat (Networking Tool) 
• A simple tool for sending and receiving data over network connections.  
• Used for port scanning, chatting between systems, and debugging.  
• Known as the “Swiss army knife of networking”.  
Nmap (Scanning Tool) 
• A tool used to discover hosts and open ports in a network.  
• Helps in security auditing and vulnerability detection.  
• Can perform port scan, OS detection, and service version detection.

11. Full Question 
Write a Python code using Scapy for creating a Network Scanner (Hint: Create an 
ARP packet using ARP() method). 
Important Terms 
• Network Scanner → Finds active devices 
• ARP (Address Resolution Protocol) → Maps IP to MAC 
• Broadcast → Sends packet to all devices 
Output 
• List of active IP addresses 
Example: 
192.168.1.1 
192.168.1.5 
192.168.1.10 
Demo 
from scapy.all import * 
arp = ARP(pdst="192.168.1.0/24") 
ether = Ether(dst="ff:ff:ff:ff:ff:ff") 
packet = ether/arp 
result = srp(packet, timeout=2)[0] 
for sent, received in result: 
print(received.psrc) 
What you say 
“I created a network scanner using ARP requests. The packet is broadcast to all 
devices in the network. Active devices reply with their IP address. This helps identify 
which hosts are currently active.”
